## Instalação (local)

1. Realize o clone do repositório:

~~~~
https://github.com/Jailsonsdsj/projeto-ES-front.git
~~~~

2. Execute o seguinte comando no terminal (necessário node.js):

~~~~

~~~~

3. Por fim, execute o app em modo de desenvolvimento:
